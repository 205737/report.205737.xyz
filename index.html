2. Investigation


I knew that I wanted to base my data around networking in some sort of way. I am currently on the waiting list to get fibre. This inspired me to look into how the network speed across Ireland has been affected by the roll out of fibre.


After a quick peep, I knew I had enough data for my project and to conduct my data analysis on.

3. Plan & Design


Goals:

  • I will have charts that show how the rollout of fibre broadband has affected the average speed for the consumer.
  • I will have charts that can be interacted with to help analyze the data further.
  • I will get my data from a source like NBI so that I can not only trust it but so it would be publicly available.
  • I will use GitHub to host my website.
